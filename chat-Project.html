コンセプト

NKCの学生が気軽に使うチャットサイト

制作の目的

動的サイトを作る

制作人数

5人

制作期間

1ヶ月

使用ツール

ペイント・ペイント3D / scrapbox / Figma / Excel

担当範囲

Design / HTML・CSS

制作ポイント・工夫

メンバー間で効率的な情報共有を実現するためにscrapboxとfigmaを導入しました。 グループチャットのお気に入り登録や、コメントしたチャットルームを一覧表示したり、既存のチャットルームを検索できるなど、ユーザーが求める機能を開発して、使いやすく便利なサイトに仕上げました。

コメント

セキュリティーに力を注ぎ、悪意あるデータの送信を防ぐためにエスケープ処理などを実装しました。これにより、安全性を確保しつつ、ユーザーが安心してサイトを利用できる環境を構築しました。
